# The-Gem-Hotel

Gem-Hotel is a website built with React, TypeScript, and Tailwind CSS. It utilizes React Context for state management and consists of four main pages: Home, Rooms, Single Room, and Contact Us.

## Features

- Landing page showcasing featured rooms
- Rooms page with filter options by price, pet allowance, and room capacity
- Single Room page displaying detailed information about an individual room
- Contact Us page for getting in touch with the hotel

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/K-Othman/The-Gem-Hotel.git
   ```

2. Navigate to the project directory:

   ```bash
   cd gem-hotel
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

## Usage

1. Start the development server:

   ```bash
   npm run dev
   ```

2. Explore the different pages of the website through the navigation menu.

3. On the Rooms page, you can utilize the filter options to narrow down your search:

- Price: Use the arrows to sort them by price.
- Pet Allowance: Choose between "Allowed" and "Not Allowed" to filter rooms based on pet policies.
- Room Capacity: Select the desired capacity from the dropdown to view rooms that accommodate the specified number of guests.


## Dependencies

The main dependencies used in this project are:

- React
- React Router DOM
- Tailwind CSS
- Vite
